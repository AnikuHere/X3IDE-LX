# Current use state
 The UI for the app isnt supported on Linux - it is already very buggy.

# Installation
 The Linux version does not have a installer. 
 Steps for installation:
 1) You have to download the X3IDE_LX-FORK.py file
 2) Allow the .PY file to be executed as program: chmod +x /path/to/your/x3ide/.py/file
 3) Double click the .py file and done, now you can use the app.
